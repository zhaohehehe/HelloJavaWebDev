(1)方法一:用相应的HTML实体(&lt;)替换所有小于号等其他符号 

<script type="text/javascript">
function compare1(a,b){
	if(a < b){
		alert("A小于B");
	}
	else if(a>b){
		alert("A大于B");
	}
	else{
		alert("A等于B");
	}
}
</script>

(2)方法二：CData片段包含js代码 

<script type="text/javascript">
<![CDATA[
         function compare2(a,b){
        	 if(a < b){
        		 alert("A小于B");
        		 }
        	 else if(a>b){
				 alert("A大于B");
				 }
        	 else{
				 alert("A等于B");
				 }
        	 }
]]>
</script>

(3)方法三(完美写法)：因为很多浏览器不兼容XHTML，因而不兼容CData片段，所以用js注释将cData标记注释掉即可 

<script type="text/javascript">
//<![CDATA[
         function compare3(a,b){
        	 if(a < b){
        		 alert("A小于B");
        		 }
        	 else if(a>b){
				 alert("A大于B");
				 }
        	 else{
				 alert("A等于B");
				 }
        	 }
//]]>
</script>
