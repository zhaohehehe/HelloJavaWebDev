操作符

  1. 一元操作符

  2. 位操作符：对特殊的NaN和Infinity值应用位操作时，这两个值都会被当做零处理

    1. 按位非（NOT）

    2. 按位与（AND）

    3. 按位或（OR）

    4. 按位异或（XOR）

    5. 左移（用 0 填充）

    6. 有符号的右移（用 符号位 的值填充）

    7. 无符号的右移（用 0 填充）

  3. 布尔操作符

    1. 逻辑非（NOT）:返回布尔值

    2. 逻辑与（AND）:可以应用于任何类型的操作数，不仅仅是布尔值

    3. 逻辑或（OR）:也可以应用于任何类型的操作数，不仅仅是布尔值

  4. 乘性操作符

    1. 乘法

    2. 除法

    3. 求模

  5. 加性操作符

    1. 加法

    2. 减法

  6. 关系操作符

  7. 相等操作符:相等和不相等(==,!=)：强制转型再比较;
                          全等和不全等(===,!==)：直接比较，不强制转型

  8. 语句

    1. do-while

    2. for-in用于遍历数组（遍历的是数组的每一项，返回每一项的toString()方法。对于普通数组返回的是每项的字符串值，对于对象数组则返回的 是Object）或对象（遍历的是对象的属性）

    3. label 和 for(break,continue)合用效果

    4. with(不建议使用！！)

    5. Switch(全等符操作！)

  9. 函数

    1. return + 同名函数会被覆盖

    2. 命名参数只提供便利，不是必需的，可以通过arguments对象访问参数数组

    3. ECMAScript没有重载，可以通过arguments.length功能模拟重载

    4. 命名参数可以和arguments对象互换使用

    5. arguments对象中的值会映射到相应的命名参数（二者同步），即使二者的内存空间不同（相互独立）最好不这么做，建议不要试图修改或重写arguments对象
