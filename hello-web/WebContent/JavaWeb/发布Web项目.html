1. 推荐部署方法：不需要拷贝项目，效率高

    1. 配置web.xml文件<Context docBase="D:\Users\zhaohe\Documents\workspace\workspace_JavaW\HelloWeb\WebContent" path="/HelloWeb" reloadable="true"/>

  2. 将项目Export成war包

    1. 将项目Export成war包，将war包放到tomcat安装路径的webapps路径下
    2. 在tomcat安装路径下的conf/server.xml文件中 <Host>标签中添加如下代码： 
      	<Context docBase="C:\Program Files\Java\tomcat-8.0.23\webapps\HelloWeb.war" path="/HelloWeb" reloadable="true"/>
      (1)运行startup.bat，启动tomcat，就可以在浏览器地址中输入：http://localhost:8080/HelloWeb/index.jsp，访问应用HelloWeb的首页;
      注意：可能出现HTTP Status 500 - java.lang.IllegalStateException: No output folder error错误
        这是因为是因为执行tomcat的startup.bat的用户的权限不够导致的，解决办法如下：
      1. 找到tomcat安装文件，如：tomcat-8.0.23，右击文件，选择 属性-->安全-->编辑；
      2. 选择Users权限，允许完全控制，之后重新启动tomcat，成功！
      3. 之后也可以将权限改回来，对运行不会再有影响（Users默认权限为：读取和执行+列出文件夹内容+读取）
      (2)配置是为了在index.jsp中配置 <a>标签的href="/HelloWeb/Demo/jsonDemo/jsonDemo_1.html"等价于
        href="<%=path %>/Demo/jsonDemo/jsonDemo_1.html"
      不配置会找不到资源。
  3. HTTPステータス 500 - Error instantiating servlet class com.shengsiyuan.study.servlet.LoginServlet
    1. 右击项目-->Build Path，将Default Output Folder改为：HelloWebWorld/WebContent/WEB-INF/classes
    2. 重启服务器
